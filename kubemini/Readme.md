This Project is about deploying K8s with Mongo-Express as Frontend and Mongo as Database.
++++++++++++++++++++++++++++++++++++++
```
minikube start
kubectl get pod
kubectl apply -f kubemini
kubectl get pod
kubectl get configmap
kubectl get secret
kubectl get svc
minikube ip
kubectl get node -o wide
kubectl delete -f kubemini
```
